Python Lists

Complete guide to Python's list data structure

List Basics

Lists are ordered, mutable collections that can contain mixed data types. They are one of Python's most versatile data structures, allowing you to store and manipulate sequences of items.

# List creation and indexing
fruits = ["apple", "banana", "cherry"]
print(fruits[0])    # Output: apple
print(type(fruits)) # Output: 
print(fruits)       # Output: ['apple', 'banana', 'cherry']

Key characteristics:

  • Zero-based indexing: The first element is at position 0 (e.g., fruits[0]).
  • Can contain duplicates: Multiple identical elements are allowed (e.g., ["apple", "apple"]).
  • Mutable: Elements can be modified after creation (e.g., fruits[0] = "pear").
  • Mixed types: Can hold different data types (e.g., [1, "text", True]).

# List operation examples
fruits = ["apple", "banana"]
fruits.append("cherry")        # fruits = ["apple", "banana", "cherry"]
fruits.insert(1, "mango")      # fruits = ["apple", "mango", "banana", "cherry"]
fruits.remove("banana")        # fruits = ["apple", "mango", "cherry"]
fruits.pop()                   # fruits = ["apple", "mango"]
fruits.extend(["kiwi", "pear"]) # fruits = ["apple", "mango", "kiwi", "pear"]
print(len(fruits))             # Output: 4

List Slicing

List slicing allows you to access sublists using the syntax [start:stop:step], where start is inclusive and stop is exclusive.

# List slicing examples
numbers = [0, 1, 2, 3, 4, 5]
print(numbers[1:4])    # Output: [1, 2, 3]
print(numbers[::2])    # Output: [0, 2, 4]
print(numbers[-2:])    # Output: [4, 5]
print(numbers[::-1])   # Output: [5, 4, 3, 2, 1, 0]
