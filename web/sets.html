Python Sets

Complete guide to Python's set data structure

Set Basics

Sets are unordered, mutable collections of unique elements. They are ideal for storing items where duplicates are not allowed and order does not matter.

# Set creation
myset = {"apple", "banana", "cherry"}
print(myset)        # Output: {'apple', 'banana', 'cherry'} (order may vary)
print(type(myset)) # Output:

Key characteristics:

  • Unordered: Elements have no index or fixed position (e.g., myset[0] raises an error).
  • Unique elements: Duplicates are removed (e.g., {"apple", "apple"} becomes {"apple"}).
  • Mutable: Elements can be added or removed, but elements must be immutable (e.g., strings, numbers, tuples).
  • Mixed types: Can hold different immutable types (e.g., {"apple", 42, True}).

Sets support various methods and operations for manipulating elements and performing mathematical set operations:

# Set operation examples
myset = {"apple", "banana"}
myset.add("cherry")           # Adds 'cherry'
myset.update(["orange", "apple"]) # Adds 'orange' (ignores duplicate 'apple')
myset.discard("banana")       # Removes 'banana'
print(len(myset))            # Output: 3
set1 = {"apple", "banana", "cherry"}
set2 = {"banana", "cherry", "date"}
print(set1.union(set2))       # Output: {'apple', 'banana', 'cherry', 'date'}
print(set1.intersection(set2)) # Output: {'banana', 'cherry'}
