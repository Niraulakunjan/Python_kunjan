Python Dictionaries

Complete guide to Python's dictionary data structure

Dictionary Basics

Dictionaries are ordered (since Python 3.7), mutable collections of key-value pairs. They are ideal for storing data with unique keys for quick lookups.

# Dictionary creation
mydict = {"name": "Alice", "age": 25, "city": "New York"}
print(mydict)        # Output: {'name': 'Alice', 'age': 25, 'city': 'New York'}
print(type(mydict))  # Output:

Key characteristics:

  • Ordered: Maintains insertion order (e.g., mydict["name"] accesses "Alice").
  • Unique keys: Duplicate keys are not allowed; later values overwrite earlier ones (e.g., {"name": "Alice", "name": "Bob"} becomes {"name": "Bob"}).
  • Mutable: Key-value pairs can be added, modified, or removed.
  • Mixed types: Keys must be immutable (e.g., strings, numbers, tuples); values can be any type.

# Dictionary operation examples
mydict = {"name": "Alice", "age": 25}
mydict["city"] = "New York"       # Adds 'city': 'New York'
mydict.update({"age": 26, "country": "USA"}) # Updates 'age', adds 'country'
print(mydict.get("name"))         # Output: Alice
print(len(mydict))                # Output: 3
print(mydict.keys())              # Output: dict_keys(['name', 'age', 'city', 'country'])
print(mydict.values())            # Output: dict_values(['Alice', 26, 'New York', 'USA'])
mydict.pop("city")                # Removes 'city': 'New York'
