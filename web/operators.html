Python Operators

Learn about different types of operators in Python

Arithmetic Operators

OperatorDescriptionExample
+Additionx + y
-Subtractionx - y
*Multiplicationx * y
/Divisionx / y
%Modulusx % y
**Exponentiationx ** y
//Floor divisionx // y
# Arithmetic operator examples
x = 10
y = 3
print(x + y)    # Output: 13
print(x - y)    # Output: 7
print(x * y)    # Output: 30
print(x / y)    # Output: 3.3333333333333335
print(x % y)    # Output: 1
print(x ** y)   # Output: 1000
print(x // y)   # Output: 3

Comparison Operators

OperatorDescriptionExample
==Equalx == y
!=Not equalx != y
>Greater thanx > y
<Less thanx < y
>=Greater than or equalx >= y
<=Less than or equalx <= y
# Comparison operator examples
x = 5
y = 8
print(x == y)   # Output: False
print(x != y)   # Output: True
print(x > y)    # Output: False
print(x < y)    # Output: True
print(x >= y)   # Output: False
print(x <= y)   # Output: True

Logical Operators

OperatorDescriptionExample
andReturns True if both statements are truex < 5 and x < 10
orReturns True if one of the statements is truex < 5 or x < 4
notReverse the resultnot(x < 5 and x < 10)
# Logical operator examples
x = 3
print(x < 5 and x > 0)      # Output: True
print(x < 5 or x > 10)      # Output: True
print(not(x < 5))           # Output: False
