Python Functions

Creating reusable code blocks in Python

Function Definition

Functions are defined using the def keyword:

def greet(name):

return f"Hello, {name}!"

Key components:

  • Parameters (inputs)
  • Docstring (optional documentation)
  • Function body
  • Return statement

Function Arguments

Python supports different argument types:

def describe_pet(pet_name, animal_type="dog"):

print(f"I have a {animal_type} named {pet_name}")

Types of arguments:

  • Positional arguments
  • Keyword arguments
  • Default parameters
  • Variable-length arguments (*args, **kwargs)
