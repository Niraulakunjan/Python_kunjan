Python Variables Tutorial

Variable Names

In Python, variables are containers for storing data values. Here are the rules for Python variables:

  • Must start with a letter or underscore
  • Can contain letters, numbers, and underscores
  • Case-sensitive (age, Age, and AGE are different variables)
# Valid variable names
name = "John"
_age = 25
user_score = 100
TotalAmount = 999.99

# Invalid variable names
2name = "John"    # Cannot start with a number
user-score = 100  # Cannot use hyphen
class = "Python"  # Cannot use reserved keywords

Assign Multiple Values

Python allows you to assign values to multiple variables in one line:

# Multiple assignment
x, y, z = 1, 2, 3

# Same value to multiple variables
a = b = c = 0

# Unpacking a list
fruits = ["apple", "banana", "cherry"]
x, y, z = fruits

Output Variables

Python has several ways to output variables:

# Basic print
name = "Alice"
age = 25
print(name)
print("My name is", name)

# String formatting
print(f"I am {name} and I am {age} years old")
print("I am {} and I am {} years old".format(name, age))

# Concatenation
print("Name: " + name + ", Age: " + str(age))

Global Variables

Variables created outside functions are global variables:

# Global variable
global_var = "I am global"

def my_function():
    # Using global keyword
    global global_var
    print(global_var)
    
    # Local variable
    local_var = "I am local"
    print(local_var)

my_function()
print(global_var)  # Works
# print(local_var)  # This would cause an error

Variable Types in Python

# Numeric Types
int_num = 10
float_num = 10.5
complex_num = 3 + 4j

# Text Type
name = "Python"

# Boolean Type
is_valid = True

# Sequence Types
my_list = [1, 2, 3]
my_tuple = (1, 2, 3)
my_range = range(6)

# Mapping Type
my_dict = {"name": "John", "age": 30}

# Set Types
my_set = {1, 2, 3}
my_frozenset = frozenset({1, 2, 3})

# None Type
my_none = None

# Check type of variable
print(type(int_num))    # 
print(type(name))       # 
print(type(my_list))    #
