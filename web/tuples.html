Python Tuples

Complete guide to Python's tuple data structure

Tuple Basics

Tuples are ordered, immutable collections that can contain mixed data types. They are similar to lists but cannot be modified after creation, making them suitable for fixed data.

# Tuple creation and indexing
mytuple = ("apple", "banana", "cherry")
print(mytuple[0])    # Output: apple
print(type(mytuple)) # Output: 
print(mytuple)       # Output: ('apple', 'banana', 'cherry')

Key characteristics:

  • Zero-based indexing: The first element is at position 0 (e.g., mytuple[0]).
  • Can contain duplicates: Multiple identical elements are allowed (e.g., ("apple", "apple")).
  • Immutable: Elements cannot be modified after creation (e.g., mytuple[0] = "pear" raises an error).
  • Mixed types: Can hold different data types (e.g., (1, "text", True)).

Tuple Operations

Tuples support limited methods due to their immutability, but operations like concatenation and repetition are available:

OperationDescriptionExample
len()Get length of tuplelen(mytuple) → 3
count()Count occurrencesmytuple.count("apple") → 2
index()Get index of itemmytuple.index("banana") → 1
+Concatenate tuplesmytuple + ("orange",)
*Repeat tuplemytuple * 2
# Tuple operation examples
mytuple = ("apple", "banana", "apple", "cherry")
print(len(mytuple))           # Output: 4
print(mytuple.count("apple")) # Output: 2
print(mytuple.index("banana")) # Output: 1
newtuple = mytuple + ("orange",) # Output: ('apple', 'banana', 'apple', 'cherry', 'orange')
print(newtuple * 2)           # Output: ('apple', 'banana', 'apple', 'cherry', 'orange', 'apple', 'banana', 'apple', 'cherry', 'orange')

Tuple Slicing

Tuple slicing allows you to access subtuple using the syntax [start:stop:step], where start is inclusive and stop is exclusive.

# Tuple slicing examples
numbers = (0, 1, 2, 3, 4, 5)
print(numbers[1:3])    # Output: (1, 2)
print(numbers[::2])    # Output: (0, 2, 4)
print(numbers[-2:])    # Output: (4, 5)
print(numbers[::-1])   # Output: (5, 4, 3, 2, 1, 0)
