Python If-Else

Complete guide to Python's conditional statements

If-Else Basics

Conditional statements in Python (`if`, `elif`, `else`) allow programs to make decisions based on conditions. They evaluate expressions to `True` or `False` and execute corresponding code blocks.

# Basic if-else example
age = 18
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")
# Output: You are an adult.

Key characteristics:

  • Indentation: Code blocks under `if`, `elif`, and `else` must be indented (typically 4 spaces).
  • Boolean evaluation: Conditions must evaluate to `True` or `False` (e.g., age >= 18).
  • Optional clauses: `elif` and `else` are optional; `if` can stand alone.
  • Nested conditions: `if` statements can be nested for complex logic.

If-Else Examples

Conditional statements support various patterns for decision-making:

PatternDescriptionExample
ifExecute if condition is Trueif x > 0: print("Positive")
if-elseExecute one of two blocksif x % 2 == 0: print("Even") else: print("Odd")
if-elif-elseCheck multiple conditionsif x > 0: print("Positive") elif x < 0: print("Negative") else: print("Zero")
Nested ifConditions within conditionsif x > 0: if x < 10: print("Single digit")
and/orCombine conditionsif x > 0 and x < 100: print("In range")
# If-elif-else example
score = 85
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")
# Output: Grade: B

# Nested if example
number = 15
if number > 0:
    if number % 2 == 0:
        print("Positive even number")
    else:
        print("Positive odd number")
# Output: Positive odd number
