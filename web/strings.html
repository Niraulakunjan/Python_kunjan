Python String Data Type

Overview of String Data Type

Text Type

  • str (string)

String Operations

Slicing Strings

# String slicing examples
text = "Hello, World!"
print(text[0:5])      # Output: Hello
print(text[7:])       # Output: World!
print(text[-6:-1])    # Output: World
print(text[::2])      # Output: Hlo ol!

Modify Strings

# String modification examples
text = "  Hello, World!  "
print(text.upper())       # Output:   HELLO, WORLD!  
print(text.lower())       # Output:   hello, world!  
print(text.strip())       # Output: Hello, World!
print(text.replace("H", "J"))  # Output:   Jello, World!

Concatenate Strings

# String concatenation examples
first = "Hello"
second = "World"
greeting = first + " " + second
print(greeting)           # Output: Hello World
print(first * 3)          # Output: HelloHelloHello
print(f"{first}, {second}!")  # Output: Hello, World!

Format Strings

# String formatting examples
name = "Alice"
age = 25
print(f"My name is {name} and I am {age}.")  # Output: My name is Alice and I am 25.
print("My name is {} and I am {}.".format(name, age))  # Output: My name is Alice and I am 25.
print("My name is %s and I am %d." % (name, age))      # Output: My name is Alice and I am 25.

Escape Characters

# Escape character examples
text = "She said, \"Hello!\""
print(text)               # Output: She said, "Hello!"
print('It\'s a test')     # Output: It's a test
print("Line 1\nLine 2")   # Output: Line 1
                          #         Line 2
print("Tab\tSpace")       # Output: Tab   Space

String Methods

# String method examples
text = "Hello, World!"
print(text.startswith("He"))    # Output: True
print(text.endswith("!"))       # Output: True
print(text.find("World"))       # Output: 7
print(text.count("l"))          # Output: 3
print(text.split(","))          # Output: ['Hello', ' World!']
print(text.isalpha())           # Output: False (due to comma and space)
